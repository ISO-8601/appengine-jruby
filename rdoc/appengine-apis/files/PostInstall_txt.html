To use the local datastore you need to have the Google App Engine SDK for Java installed (code.google.com/appengine/docs/java/gettingstarted/installing.html). Then you need to add several jars to your classpath.

For example: $ export SDK=`pwd`/appengine-java-sdk/lib $ export CLASSPATH=$SDK/shared/appengine-local-runtime-shared.jar:$SDK/impl/appengine-api-stubs.jar:$SDK/impl/appengine-api.jar:$SDK/impl/appengine-local-runtime.jar

For more information on appengine-apis, see code.google.com/p/appengine-jruby
