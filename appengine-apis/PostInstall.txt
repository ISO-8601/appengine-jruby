
To use the local datastore you need to have the Google App Engine SDK for Java
installed
(http://code.google.com/appengine/docs/java/gettingstarted/installing.html).
Then you need to add several jars to your classpath.

For example:
export CLASSPATH=appengine-java-sdk/lib/shared/appengine-local-runtime-shared.jar:appengine-java-sdk/lib/impl/appengine-api-stubs.jar:appengine-java-sdk/lib/impl/appengine-api.jar:appengine-java-sdk/lib/impl/appengine-local-runtime.jar

For more information on appengine-apis, see
http://code.google.com/p/appengine-jruby
